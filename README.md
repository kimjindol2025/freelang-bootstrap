## 개요

FreeLang의 기초가 되는 최소한의 부트스트랩 구현. 핵심 기능만 포함하여 다른 언어 구현의 기반으로 사용 가능.

---

## 🎯 포함된 기능

### 핵심
- ✅ 렉서(Lexer)
- ✅ 파서(Parser)
- ✅ 기본 인터프리터
- ✅ 변수 & 함수

### 표준 기능
- ✅ 산술 연산
- ✅ 논리 연산
- ✅ 제어 흐름 (if/for)
- ✅ 기본 입출력

---

## 🚀 시작하기

```bash
# 컴파일
make build

# 실행
./bootstrap examples/hello.fl

# 대화형 모드
./bootstrap --repl
```
